About the Financial Solution Predictor

Cuurently the main purpose of this app is to represent a workflow system that simulates an advisor's screening process. The action is presumed to start when an advisor has most of the client's information is finding a portfolio that might match the client.

The following system will consider the following parameters:

  • Risk Profile
  • Risk Tolerance
  • Risk Rating

The system should work as follows: Take in this input and then convert it into a structured risk score, then map the scores to the following ASISA categories.
